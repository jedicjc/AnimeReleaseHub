# AnimeReleaseHub

AnimeReleaseHub is a platform for tracking upcoming anime, release dates, English dubs, and anime news.

Powered by MapleOS.

## Project Areas

- backend: API server
- frontend: website
- mapleos: anime news intelligence system
- discord-bot: Discord announcements
- database: database schema and migrations
- docs: project documentation
- docker: Docker setup
